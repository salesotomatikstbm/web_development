Web Development Basics

Let's learn how websites work!

What is a Website?

A website is like a digital book that lives on the internet. Instead of paper pages, it has web pages made with code.

Example: Your school website might have:
  • A homepage (like a book cover)
  • An about page (like the "about the author" section)
  • A contact page (like the back cover with address)
Try This: Open your favorite website and count how many different pages it has.

Three Building Blocks

Every website is made with three special languages that work together:

1. HTML - The Skeleton
Creates the structure (headings, paragraphs, images)
Example: <h1>My Pet</h1> <p>I have a brown dog.</p>

2. CSS - The Clothes
Makes it look nice (colors, fonts, layout)
Example: h1 { color: blue; } p { font-size: 18px; }

3. JavaScript - The Brain
Makes it interactive (buttons, forms, animations)
Example: button.addEventListener('click', function() { alert('Hello!'); });
Imagine building a robot:
HTML = the metal frame
CSS = the paint and clothes
JavaScript = the computer that makes it move

How Do Websites Reach You?

When you type a website address (like www.google.com):

  1. Your browser (Chrome, Firefox) sends a request
  2. A server (powerful computer) sends back the HTML, CSS, and JavaScript files
  3. Your browser puts them together like puzzle pieces
  4. You see the complete website!
Real-life Example:
Ordering pizza:
1. You call the pizza place (browser request)
2. They prepare your pizza (server processes)
3. Delivery brings it (files sent to you)
4. You eat the pizza (see the website)

Let's See Real Code!
